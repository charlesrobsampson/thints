Color System Reference

HUE 0–9 · LIGHTNESS 0–4 · SATURATION 5–9 · GRAYSCALE 0–9

Hue Scale — integers + key decimals (default: lightness 2, saturation 9)
Lightness Scale (0–4) — using hue 4 (Green) at full saturation
Saturation Scale (5–9) — using hue 4 (Green) at default lightness (2)
Combined Examples — hue · lightness · saturation
Default Value Examples — what omitting a parameter implies

Hue only → lightness 2 (mid), saturation 9 (pure) assumed.
Hue + saturation → lightness 2 (mid) assumed. Saturation written as 5–9.
Hue + lightness → saturation 9 (pure) assumed. Lightness written as 0–4.
Saturation/lightness only (no hue) → use grayscale convention instead.

Grayscale (separate convention — 0 black → 9 white)

Grayscale uses a dedicated keyword — no hue or saturation. gray 0 = black, gray 9 = white, gray 4-5 = mid gray.
